# Screenshots

Place UI screenshots here. Suggested filenames (referenced in the root README):

| Filename | Content |
|----------|---------|
| `login.png` | Login screen |
| `chat.png` | Main chat interface with an active query and answer |
| `sources.png` | Source citations panel expanded |
| `upload.png` | Document upload dialog |
| `admin.png` | Admin panel — user management or corpus status tab |
| `pii.png` | PII scanner results on an uploaded document |

Screenshots can be captured by running the app locally:

```
streamlit run Code/app.py
```

Then navigate to `http://localhost:8501` and use the browser's screenshot tool or a tool like Snipping Tool (Windows) / Screenshot (macOS).
